Dotly

Developed by Diego Cuentas

Description

Dotly is a graphical tool built in Python that allows generating dotplots from two FASTA files. Quickly and intuitively visualize similarities and differences between sequences.

Quick Tutorial

  1. Open the program by running main.py in Python.
  2. Click the "Load FASTA" buttons to select your two FASTA files from your computer.
  3. Verify that both sequences are loaded in the interface.
  4. Click "Generate DotPlot" to visualize the comparison. You can adjust the window size and stringency parameters to modify the sensitivity of the dotplot.
  5. Click "Download" to save the resulting dotplot image.
